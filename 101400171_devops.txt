COMP3104-DevOps: 
The DevOps course teaches about effective DevOps techniques and being able to provide solutions using industry standard tools and techniques. 
The course also teaches about using a version control system for low-risk software releases.  
Other course skills that are taught in COMP3104 include Github actions CI/CD, automated testing, code quality checks and infrastructure as Code(IaC).

DevOps combines development (Dev) and operations (Ops) to increase the efficiency, speed, and security of software development and delivery compared to traditional processes. A more nimble software development lifecycle results in a competitive advantage for businesses and their customers.
