DevOps combines development (Dev) and operations (Ops) to increase the efficiency, speed, and security of software development and delivery compared to traditional processes. A more nimble software development lifecycle results in a competitive advantage for businesses and their customers.

In week 1, I learned about the basics of DevOps, build pipelines and different types of bash commands

In week 2, I learned about what GitHub is and how to use the git commands

In week 3, I learned about version control systems and what Travis CI is 

In week 4, I learned about the tools and techniques and what a DevOps Engineer is

In week 5, I learned about the single page application, the pros and cons and the build tools for DevOps
