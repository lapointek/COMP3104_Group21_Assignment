COMP3104-DevOps: 
COMP3104 DevOps course teaches about effective DevOps techniques and being able to provide solutions using industry standard tools and techniques. 
The course also teaches about using a version control system for low-risk software releases.  
Other course skills that are taught in COMP3104 include Github actions CI/CD, automated testing, code quality checks and infrastructure as Code(IaC).

DevOps Tools:
Plan: Jira, Trello, Confluence
Code: Github, Gitlab, Bitbucket, Visual Studio Code
Build: Docker, Maven, Gradle, Ant
Test: Selenium, JUnit, Postman
Release: Jenkins, Bamboo
